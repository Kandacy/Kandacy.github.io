## 文档介绍		

​		文档中列出了绝大部分的配置项。大部分易于理解的配置项直接用表格展示。少数易混淆、晦涩难懂的配置会用长文本介绍。

​		未列出的配置项有如下三种情况。请读者自行斟酌。

- 一眼就看得懂的（例如某些开关配置）
- 不起作用的（例如某些“字间距”配置）
- 忘了写（QAQ）
